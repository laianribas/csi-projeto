# Front-end

## :computer: Instalação e execução: passo-a-passo:

Clone o projeto

```bash
  git clone https://github.com/laianribas/csi-projeto.git
```

Entre no diretório do front-end do projeto

```bash
  cd csi-projeto/frontend
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
